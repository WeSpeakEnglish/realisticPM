# RealisticPM
**RealisticPM** is a on-wall air quality monitor, powered via USB-C connector

**Features:**
-Big color display
-Standard USB-C connector
-1.5W power supply
-High Quality LEDs
-COM port data output (via virtual COM port)
-Open source code
